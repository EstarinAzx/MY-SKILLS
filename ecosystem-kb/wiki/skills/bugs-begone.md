# bugs-begone

Interactive debugging mode for hard-to-diagnose bugs: generates hypotheses, instruments code with runtime logs, fixes iteratively with human-in-the-loop verification. Explicitly opt-in — the agent reminds the user it exists but never auto-activates it. For routine bugs, [[superpowers]]' `systematic-debugging` fires instead.
