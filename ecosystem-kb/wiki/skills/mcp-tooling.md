# mcp-tooling

Skills for working *with* MCP servers (the servers themselves are wired in [[mcp-servers]]):

- **mcp2cli** — convert any MCP server into standalone Bash-invokable scripts (MCP → CLI wrapper).
- **unity-mcp-skill** — orchestrate Unity Editor via MCP for Unity: GameObjects, scripts, scenes, tests. Best practices + tool schemas.
- **mcp-source** — `/mcp-source [main|beta|branch|local]`: swap the MCP-for-Unity package source in connected Unity projects between upstream releases, remote branch, or local dev checkout.
