# design-skills

The frontend/design skill family after the 2026-06-11 audit ([[design-skill-lineup]]). Routing:

- **impeccable** — primary design engine (v3 of Anthropic frontend-design; ~30 reference files + live browser scripts). Route ALL frontend design/redesign/critique/polish/a11y/motion/typography work here first.
- **redesign-skill** — upgrading an *existing* site/app to premium quality without breaking function; audits for generic AI patterns.
- **minimalist-skill** — deliberate aesthetic preset: clean editorial, warm monochrome, flat bento, no gradients.
- **brutalist-skill** — deliberate aesthetic preset: Swiss-print × military-terminal, rigid grids, analog degradation.
- **brandkit** — image generation: brand-guideline boards, logo systems, identity decks.
- **imagegen-frontend-web** — image generation: one horizontal reference image *per section* of a landing page.
- **imagegen-frontend-mobile** — image generation: premium app-native mobile screens in phone mockups; images only, no code.

Deprecated to `~/.claude/_deprecated/`: taste-skill, soft-skill, gpt-tasteskill, stitch-skill, image-to-code-skill — don't reinstate.
